# Modulo de patrones y riesgo

## Proposito

Detectar patrones repetitivos con ventanas deslizantes y clasificar activos por volatilidad.

## Archivos relacionados

- `analisis_financiero.py`
- `api.py`
- `static/modules/patterns/index.html`

## Elementos clave

- Racha alcista.
- Rebote tras caida.
- Volatilidad anualizada.
- Categoria de riesgo.

## Evidencia para sustentar

- Formula de retorno diario.
- Desviacion estandar muestral.
- Volatilidad anualizada.
- Ordenamiento del ranking de riesgo.
