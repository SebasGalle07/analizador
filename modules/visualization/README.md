# Modulo de visualizacion y PDF

## Proposito

Presentar los resultados analiticos mediante graficas y exportarlos en un reporte tecnico.

## Archivos relacionados

- `src/visualizacion.py`
- `src/reporte_pdf.py`
- `api.py`
- `static/modules/visualization/index.html`

## Visualizaciones

- Heatmap de correlacion.
- Grafico de velas con SMA.
- Serie comparativa de precios.
- Retornos diarios.
- Barras de riesgo.

## Evidencia para sustentar

- Que representa cada grafico.
- Que formulas utiliza.
- Que algoritmo lo construye.
- Como se inserta en el PDF.
