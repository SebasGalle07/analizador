# Modulo de similitud

## Proposito

Comparar dos activos financieros usando algoritmos clasicos implementados a mano.

## Archivos relacionados

- `analisis_financiero.py`
- `visualizacion.py`
- `api.py`
- `static/modules/similarity/index.html`

## Algoritmos

- Distancia euclidiana.
- Correlacion de Pearson.
- Dynamic Time Warping.
- Similitud coseno.

## Evidencia para sustentar

- Alineacion de series.
- Calculo manual de retornos.
- Complejidad temporal y espacial.
- Interpretacion de resultados.
